## How to Play

1. Open your mouth (jaw open) to activate eating mode
2. Position your mouth under the falling colored circles
3. Eat 5 circles to win
4. When you win, "Happy Birthday" pops up 


## Technologies Used

- p5.js
- MediaPipe Face Landmarks
- Web Audio API
